# حاسبة احتساب التعويضات البيئية

أداة ويب تفاعلية لاحتساب قيمة التعويضات الناجمة عن مخالفات إلقاء مياه الصرف والسوائل غير المعالجة في الأوساط البيئية، المحالة إلى النيابة العامة، وفقًا لمبدأ "الملوّث يدفع" والمعادلة المرجعية الواردة في دليل احتساب التعويضات البيئية:

```
ت = ( خم + خغ ) + كت + دب − خصم
```

- **خم**: الأضرار المباشرة
- **خغ**: الأضرار غير المباشرة
- **كت**: تكلفة إعادة التأهيل (تشخيص + تنفيذ + متابعة)
- **دب**: قيمة التدهور البيئي (HEA / REA / ESA)
- **خصم**: ما نفّذه المخالف فعليًا من إعادة تأهيل موثّقة ومقبولة فنيًا

الأداة تفرض ضابط "لا تُحتسب أي قيمة لم تُدعم بشواهد موثّقة" عبر خانة "موثّق بشواهد" لكل عنصر، وتدعم تعدد المخالفات ضمن الحادثة الواحدة مع تجميع الإجمالي تلقائيًا، وتولّد تقريرًا قابلًا للطباعة/PDF بصيغة مناسبة للرفع إلى النيابة العامة.

لا حاجة لأي خادم أو قاعدة بيانات — الملفات الثلاثة (`index.html`, `style.css`, `app.js`) تعمل بالكامل داخل المتصفح، والقضايا المحفوظة تُخزَّن محليًا على جهاز المستخدم (localStorage).

## التشغيل محليًا

افتح `index.html` مباشرة في المتصفح، أو شغّل خادمًا محليًا بسيطًا:

```bash
npx serve .
```

## النشر عبر GitHub + Vercel

### ١) رفع المشروع إلى GitHub

```bash
cd compensation-calculator
git init
git add .
git commit -m "حاسبة احتساب التعويضات البيئية"
git branch -M main
git remote add origin https://github.com/<اسم-المستخدم>/compensation-calculator.git
git push -u origin main
```

(أنشئ المستودع أولًا من حسابك على github.com، فارغًا بدون README، ثم استخدم رابطه أعلاه.)

### ٢) النشر على Vercel

1. سجّل الدخول إلى [vercel.com](https://vercel.com) واختر **Add New → Project**.
2. اربط حساب GitHub الخاص بك واختر مستودع `compensation-calculator`.
3. عند إعداد المشروع اختر **Framework Preset: Other** (الموقع ثابت HTML/CSS/JS، لا يحتاج أمر بناء).
4. اترك **Build Command** و **Output Directory** فارغَين (أو Output Directory = `.`).
5. اضغط **Deploy**. سيصدر Vercel رابطًا مباشرًا (مثل `compensation-calculator.vercel.app`) يتحدّث تلقائيًا مع كل `git push` جديد على الفرع `main`.

> ملاحظة: تم ربط موصل Vercel داخل هذه المحادثة لمساعدتك في متابعة عمليات النشر وحالتها؛ أما ربط GitHub فيتم من خلال واجهة Vercel نفسها كما في الخطوة ٢.

## تنبيه

هذه الأداة احتساب استرشادي داخلي. لا تُعتمد أي قيمة تعويض نهائية إلا بعد مراجعتها من لجنة احتساب التعويضات واعتماد كامل الشواهد المؤيدة، تمهيدًا لرفعها إلى النيابة العامة استنادًا للمادة (٤٣) من نظام البيئة.
